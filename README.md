# Java ORM Framework

A **lightweight  Java ORM (Object-Relational Mapping) framework**, updated from the previous version for **improved performance and usability**.

**Features:**

- **CRUD operations**: `SELECT`, `INSERT`, `UPDATE`, `DELETE`

- **Caching**: Built-in support to speed up repeated queries

- **Simple configuration**: Minimal setup for rapid development

- **Flexible mapping**: Easily map Java objects to database tables

- **Tested on**: Oracle and PostgreSQL

Designed to simplify database access in Java applications while providing **efficient caching and modern ORM functionality**.
